This is Color Snake!

Color Snake isn't just a neat graphical effect, it's built with a simple generative algorithm that can be trained over multiple iterations.
This means that every iteration Color Snake's properties change a little bit.
If you chose to save an iteration, the next iteration's change will build on the last ones, slowly changing the look of Color Snake each time.
You can click on don't save this iteration to discard the latest set of changes.
If you don't like where Color Snake has ended up, you can click on the reset button to reset everything to their starting values.
